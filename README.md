# RiftRocket - Public Website

## 🚀 Getting Started.

Install the required dependencies:

```
npm install
```

## 🛠️ Developing

If you want to develop locally use:

```
npm run dev
```

## 🔧 Building

When you're ready to create a production build use:

```
npm install
```

```
npm run build
```

## 🌟 Code Formatting

```
npm run format
```
